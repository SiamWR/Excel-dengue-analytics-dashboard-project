# Excel-dengue-analytics-dashboard-project
Interactive Excel dashboard for analyzing dengue cases in Dhaka City, featuring patient demographics, geographic distribution, diagnostic markers, and public health insights.
## Overview

This project presents an interactive Microsoft Excel dashboard developed to analyze dengue cases in Dhaka City. The dashboard enables users to explore dengue prevalence, demographic characteristics, geographic distribution, diagnostic markers, and area-wise trends through dynamic visualizations and slicers.

The objective is to support data-driven public health decision-making by transforming raw dengue surveillance data into meaningful insights.
## Tools & Techniques

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Dashboard Design
- Healthcare Data Analytics
